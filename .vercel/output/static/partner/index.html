Partner registration

Partner registration opens January 15th 2024 at 10:00 Registration is done by using the a link on this page.

Even though all partners are considered equal, there are a few options to choose from. There is a base package that everyone gets, and then there are a few available expansions. Note that these expansions have limited availability.

Partner packages (both the standard package and the expansion options) are sold first come first served starting January 15th and as long as there are still packages left.
